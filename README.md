# TodoList 實作

[點我點我](https://victor81528.github.io/todolist/)

使用：
- Vue3 Composition API
- 元件化 Components
- Vuex
- Props傳值
- Git-Flow

JS方面著重元件與Vuex資料庫的分工\
首次嘗試使用Composition API\
Git-Flow確保開發流程順利\
在前輩的逼迫下提升支援解析度至4K

面向 device： 桌機、平板、手機\
開發用瀏覽器： Edge 91 for macOS
